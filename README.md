# homebrew-srcclr
Homebrew Tap for srcclr agent.

To install your own Homebrew, you can run the command:

`brew install --build-from-source <user>/<repo>/<formula>`

## Using your tap

To add your tap to the brew list of taps, use

`brew tap <user>/<repo>`

To install your tap, use

`brew install <formula>`

If a tap by the same name has already been installed, you will need to fully qualify your tap name when you install it:

`brew install <user>/<repo>/<formula>`
